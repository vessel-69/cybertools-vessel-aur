# cybertools-vessel — AUR Package

AUR package for [cybertools-vessel](https://github.com/vessel-69/cybertools-vessel) — the global CLI for [CyberTools API](https://www.cyber-tools.dev).

## Install (AUR)

**With yay:**
```bash
yay -S cybertools-vessel
```

**With paru:**
```bash
paru -S cybertools-vessel
```

**Manually:**
```bash
git clone https://aur.archlinux.org/cybertools-vessel.git
cd cybertools-vessel
makepkg -si
```

## Usage after install

```bash
cybtl                    # interactive TUI
cybtl recon example.com
cybtl workflow example.com --save report.json
```
